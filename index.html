Welcome to PyTalk Irc Pages

pytalk_irc 是一个基于web.py的在线IRC系统，用户可以在线聊天。

版本：v1.0 release 20140527

Startup

app将在81端口运行：

# python ./app.py 81

vps用户可以使用nohup，将app.py作为守护进程运行：

# nohup python ./app.py 81
